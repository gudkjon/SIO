Survey System

Survey system for IcelandicOnline

View on GitHub Download .zip Download .tar.gz

The Project

The system provides administrators of Icelandic Online an easy way to create surveys for the users, and receive their input.

It also provides the ability to create tests for users, with automatic grading.

Technology

The system was written in Java, with the assistance of Java Spring, Java Persistence API, Maven, Tomcat Connector and PostgreSQL, GitHub and Bootstrap.

Upcoming

In the upcoming months more functionality will be added, mainly an Android front end, and last but not least integration with Icelandic Online.
